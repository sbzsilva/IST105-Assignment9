Welcome to Cisco DNA Center Network Automation

This application demonstrates network automation using Cisco DNA Center REST API.

Features:

  • Authentication with Cisco DNA Center
  • Retrieve and display network device inventory
  • Fetch and analyze interface configurations
  • MongoDB integration for logging interactions
  • Django web interface for easy access

Getting Started:

Navigate through the menu to explore different functionalities:

  1. Authenticate - Connect to Cisco DNA Center and obtain authentication token
  2. Network Devices - List all network devices managed by DNA Center
  3. Device Interfaces - View interface details for a specific device
  4. View Logs - See MongoDB logs of all operations

About Cisco DNA Center:

Cisco Digital Network Architecture (DNA) Center is a network management and automation platform that provides centralized control of network devices, automation of network configurations and policies, analytics for monitoring and troubleshooting, and intent-based networking.
